Efforting.Tech Python Framework#

Introduction#

This is a python library intended for rapid application development using Python. Currently it is written and maintained by a single author so development will be slow.

Warning

The current state of this library is to be considered A SKETCH, that is UNSTABLE. The same goes for the DOCUMENTATION that may lag behind the library itself.

There will be more information here to come that will address the following items:

  • Procedures for developing and maintainin the code base

  • Procedures for documentation, issue tracking, planning and testing

  • Short and long term planning for the project

  • Procedures for releases, deprecation and regression

  • Procedures for promotion or demotion in feature stability chain

  • What this library is intended for

  • Examples of how to use it
