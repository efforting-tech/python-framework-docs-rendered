Example - ABC Basics#

The ABC system utilizes a hierarchial tree model. For example, if the ABC abc.media.book is registered, abc.media will automatically be registered too. And isinstance(some_instance, abc.media) will return True is some_instance has abc.media.book as a base class.

 1from efforting.tech.abc import create_abc, abc
 2
 3create_abc('media.paper')
 4create_abc('media.stone_tablet')
 5create_abc('information.source')
 6
 7class book(abc.media.paper, abc.information.source):
 8	pass
 9
10print(issubclass(book, abc.media.paper))				#True
11print(issubclass(book, abc.media.stone_tablet))			#False
12print(issubclass(book, abc.information))				#True
