internals.py#

Overview#

This module currently only contains one function. This function will be detailed here at a later time.

run_test(function, name, local_frame, index=None, skip=False, export=None, should_fail=False)#
Parameters

  • function – Class body function used as test

  • name (str) – Name of test

  • local_frame (frame) – Frame of test definition

  • index (str or int) – Index or name to identify sub test

  • skip (bool) – Skips this test

  • export – Not used

  • should_fail (bool) – Indicates this test is supposed to throw an exception.

This function executes the test in the function parameter in accordance with the parameters above.

Note

The parameter export will likely be removed soon.

Note

Later on we may add a feature to only accept a particular exception type or exception group for tests that should have an exception thrown so that we know if they failed the expected way or not.
