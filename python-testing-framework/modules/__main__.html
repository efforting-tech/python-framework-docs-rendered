__main__.py#

Overview#

This is the ingress point of this package. When the package is executed, all arguments will be interpreted as filenames of tests to run. After each test is run using load_and_run_test() a summary will be printed using print_summary().

Note

The current behavior is likely to change in the future as argument parsing is added. This is not yet planned out.

API#

load_and_run_test(filename)#
Parameters

filename (str or PurePath) – Path to the filename with the test definitions

Returns

None

Loads and runs tests defined in filename. View implementation details here.

print_summary(exit_process=False)#
Parameters

exit_process (bool) – Exit process after print

Returns

None

Prints out a summary of all executed registered tests (state.registered_tests).

If exit_process is set to True, the process will exit with returncode 0 if all executed tests succeeds or 1 if any of them fails.
